# Exercise_Unit5_website

This repository contains a biographic webpage about Linus Torvalds.  
It presents information about his life, achievements, and his role as the creator of the Linux operating system and Git.

## Technologies Used
- HTML
- CSS

## How to View the Website
1. Download or clone the repository.
2. Open the project folder.
3. Double-click the `index.html` file.
4. The webpage will open in your default web browser.
